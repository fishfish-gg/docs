# Internal API Docs

This page serves to document the API's internal routes for transparency and clarity purposes.
None of these endpoints are accessible outside the organization.



## Users

### `POST /users`

Create a user.

### `GET /users/:id`

Get a user.

### `PUT /users/:id`

Update a user.

### `DELETE /users/:id`

Delete a user.
